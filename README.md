Tutor-AI (“the extension”) is designed to provide contextual AI tutoring for users solving coding problems on supported websites.

---

## Data Collection

The extension collects **only the minimum data required** to provide its functionality, including:

- Page content related to the current coding problem
- User-written code visible on the page
- User preferences and session state stored locally

The extension **does not collect**:

- Personally identifiable information
- Browsing history outside supported sites
- Any data without explicit user interaction

---

## Data Usage

Collected data is used solely to:

- Generate AI-based tutoring responses
- Maintain user preferences and session continuity

---

## Data Storage

- Local data is stored using Chrome’s storage APIs
- Any data sent to a backend service is used only to generate tutoring responses
- No data is sold, shared, or used for advertising purposes

---

## User Control

The extension accesses page data **only when the user explicitly activates it**.  
Users can remove the extension at any time to stop all data access.
